# 2026-06-21 视频来源保留原画面音频增强

## 目标

用户选择视频作为音乐来源时，输出不再重新生成 MV 页面，而是保留原视频画面，只处理并替换音轨。

## 实现

- `VideoRenderer::render` 自动识别 `.mp4` / `.mkv` / `.webm` / `.mov` / `.avi` / `.flv` / `.wmv` 来源。
- 视频来源模式跳过背景读取、歌词解析、OpenCV 帧合成和 rawvideo 编码。
- 新增单输入 FFmpeg 路径：`-map 0:v:0 -map [aout] -c:v copy`，保留原视频流并写入 HiFi 双声道 3D 增强音轨。
- GUI 校验中视频来源只要求音乐来源和输出路径；背景图、歌词仅在普通音频生成 MV 模式必填。
- 禁止输出路径与原视频/音乐来源相同，避免覆盖原素材。

## 结果

视频文件可以当作播放器式音频处理素材使用：画面保持原样，导出速度主要取决于音频滤镜和封装，避免无意义的视频重绘。
