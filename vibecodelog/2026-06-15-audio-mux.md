# 2026-06-15 MP4 音轨复用实现日志

## 本次做了什么

- 在视频渲染流程中加入最终 MP4 音轨复用。
- 渲染器现在先输出临时无声视频：
  - `*.video_only.mp4`
- 然后调用系统 `ffmpeg` 命令行：
  - 输入临时视频
  - 输入用户选择的音乐文件
  - 复制视频流
  - 将音频转码为 AAC 320k
  - 使用 `-shortest` 保证音画长度对齐
  - 使用 `+faststart` 优化 MP4 播放加载
- 合并成功后自动删除临时无声视频。
- 更新 `README.md`：
  - 增加 `ffmpeg` 命令行依赖
  - 删除“音频复用后续补齐”的过时说明

## 验证结果

- 执行 `cmake --build build -j` 编译成功。
- 检查系统 `ffmpeg` 成功：
  - `/usr/bin/ffmpeg`
  - `ffmpeg version 8.0.1-3ubuntu2`
- VS Code 问题面板未发现错误。

## 当前输出行为

最终用户选择的输出路径会得到带音乐音轨的 MP4 文件。
